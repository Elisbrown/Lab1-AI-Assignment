### How to Run the Project
1. ### Ensure your folder looks like this:
   Lab1-AI-Assignment/
   ├── agent.py
   ├── environment.py
   ├── search_algorithms.py
   ├── agent_class.py
   └── README.md

2. ### Run the project:
   - Open your terminal/command prompt.
   - Navigate to the project folder:

    >> cd /path/to/Lab1-AI-Assignment

   - Run the `agent.py` file:
    
    >> python agent.py
